# 🎬 Netflix Movies & TV Shows - Exploratory Data Analysis

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows Dataset.

The aim is to clean the dataset, explore trends, visualize important information and generate meaningful business insights.

---

## 📂 Dataset

Dataset Source:
https://www.kaggle.com/datasets/shivamb/netflix-shows

File Used:

```
netflix_titles.csv
```

- Seaborn
- Google Colab

---

## 🔍 Business Insights

- Netflix contains more Movies than TV Shows.
- United States contributes the highest amount of content.
- TV-MA is the most common rating.
- Netflix rapidly expanded after 2015.
- Drama and International Movies dominate the platform.
