# Netflix Movies & TV Shows EDA

An Exploratory Data Analysis (EDA) on the Netflix Movies & TV Shows dataset. This project cleans the data, performs descriptive analytics, and visualizes trends and genre distributions using Pandas, Matplotlib, and Seaborn.

## Dataset
- **Name:** Netflix Movies and TV Shows
- **Source:** [Kaggle: shivamb/netflix-shows](https://www.kaggle.com/datasets/shivamb/netflix-shows)

## How to Run

1. **Get the Dataset (Kaggle.json requirement):**
   - You will need access to Kaggle to download the dataset. If you revert the notebook to use the Kaggle API, you **must** have your `kaggle.json` API token ready to upload when prompted.
   - In the notebook's current simplified state, please download the dataset manually from the source link above (Kaggle account required) and extract `netflix_titles.csv`.

2. **Run in Google Colab:**
   - Open `netflix_eda.ipynb` in Google Colab.
   - Run the first cell, which will prompt you to upload a file. 
   - Click "Choose Files" and select your extracted `netflix_titles.csv`.
   - Run the rest of the cells sequentially to perform the data cleaning and generate the visualizations!
